Type-Type-Type

About the Game
This is a keyboard word typing game where players can train their typing speed and accuracy through this game.

When the game is started, a sentence of word will appear together with a bomb start to fall from the sky. Players will start to type and complete the sentence as fast as possible. Every completed sentence will increase the speed of the falling bomb. When the bomb is fall onto the ground and exploded, the game is over. Have fun!
